# gpltool

## USAGE

```

gpltool <mode> <args...>
MODE <PREDICT> :
   -namemodel <pathfile> (Nom du fichier model)
   -datapredict <sequence> (données de prédiction)
MODE <EXPERIMENT> :
   -sizepop <val> (Taille de la population)
   -numbergen <val> (Nombre de generation)
   -stpcrit <val> (Critere d'arret de l'experience)
   -njobs <val> (Nombre de processus paralleles)
   -crossover <val> (p_crossover)
   -subtreemutation <val> (p_subtree_mutation)
   -hoistmutation <val> (p_hoist_mutation)
   -pointmutation <val> (p_point_mutation)
   -maxsamples <val> (max_samples)
   -parsimonycoefficient <val> (parsimony_coefficient)
   -randomstate <val> (random_state)
   -csvfile <pathfile> (CSV file path)
   -parammath <val> (parammath)
   -randomstate <val> (random_state)

```
