Hermetic Spellbook Truetype Font for Windows Version 1.0

2017 Iconian Fonts - Daniel Zadorozny 

http://www.iconian.com

This font comes with the following 3 versions: Regular, Italic and 3D. 


This is a dingbat font with over 90 witchcraft and sorcery symbols/runes.

Use of a character map program, such as Windows' Character Map, will be very helpful in exploring and using this font.
